# 紧急修复说明

## 问题：CopyPaste3D 导入错误

### 错误信息
```
ImportError: cannot import name 'PIPELINES' from 'mmdet3d.datasets.builder'
```

### 原因
我的疏忽导致导入路径错误。`PIPELINES` 应该从 `mmdet.datasets` 导入，而不是 `mmdet3d.datasets.builder`。

### 修复方案

**已自动修复** - 代码已更新并推送到 GitHub

修复内容：
```python
# 修改前
from mmdet3d.datasets.builder import PIPELINES

# 修改后
from mmdet.datasets import PIPELINES
```
